이 문제는 Bitmask DP라는 기법을 사용해서 경우의 수를 구합니다.
DP는 Dynamic Programming, 동적계획법을 의미하는데, 간단히 말하면 어떤 한 문제를 여러 개의 부분 문제로 쪼개고, 각 부분 문제의 결과를 저장해놓은 뒤 저장한 값들을 이용해 최종 결과를 구하는 방식입니다.
Bitmask는 어떤 정보를 비트 안에 담아서 표현하는 기법입니다. 이 문제에서는 칸이 채워져 있으면 1, 비어 있으면 0으로 표시하는데,
만약 3번째 칸부터 뒤로 5개 칸이 모두 비어 있으면 000000=0, 4번째 칸만 채워져 있으면 010000=16, 3번째 칸과 5번째 칸이 채워져 있으면 101000=40으로 어떤 칸이 채워져 있고 어떤 칸이 비어 있는지 나타냅니다.

설명에서 나오는 <<, >> 등의 기호는 비트 시프트 연산이라는 것으로,
x>>y는 'x라는 변수의 비트를 y칸만큼 오른쪽으로 이동한다'라는 의미입니다.
예를 들어 a=6=0110(2)이고, a<<1 이렇게 써 있으면 연산의 결과는 1100(2)=12가 됩니다.


설명에서 서술한 바와 같이 이 문제에서는 go(N,S)라는 함수를 만들어서 경우의 수를 구하는데,
이 때 N은 현재 칸 번호, S는 지금 칸부터 M개 칸의 상태를 bitmask를 이용해서 표현한 정보입니다.
M개 칸의 상태를 나타내는 이유는 항상 자신(N번째 칸)보다 오른쪽 아래에 있는 칸들의 상태만을 보기 위해서입니다.

그리고 맨 왼쪽 위(0번 칸)부터 반복문을 돌면서 도미노를 채워 나가며 경우의 수를 구해나가기 시작하는데,
만약 자신이 이미 채워져 있다면 경우의 수는 그냥 go(N+1,S>>1)이 됩니다. 즉 다음 칸에서의 경우의 수가 되는 것입니다.
이제 자신(N번째 칸)이 채워져 있지 않은 경우에 대해 살펴보겠습니다.
항상 왼쪽 위부터 순서대로 도미노를 채워나가기 때문에 반드시 자신의 바로 아래 칸은 비어 있게 되고, 따라서 반드시 2*1(세로) 도미노를 채울 수 있습니다.
그래서 경우의 수는 go(N+1, (S|(1<< m) )>>1)이 됩니다.)
또한 다음 칸(N+1) 또한 비어 있다면 1*2(가로)로도 도미노를 채울 수 있습니다.
그래서 경우의 수에 go(N+2,S>>2)를 더해줄 수 있게 됩니다.

//2119 최시율 제작--코드는 작년에 짠 코드입니다.
#include <stdio.h>
#include <iostream>
#include <algorithm>
#include <cstring>
#pragma warning(disable:4996)
using namespace std;
 
int n, m;
//맨 처음 칸을 0으로 두고 순서대로 번호를 붙임
/*
0 1 2 3
4 5 6 7 이런 식으로...
*/
int dp[200][1 << 15];//여기에다가 경우의 수를 저장
 
int go(int N, int S)//N번째 칸부터 m개 칸의 상태가 S(비트마스크)일 때 경우의 수
{
    if (N == n * m&& S == 0)//마지막 칸에서 S가 0=>아래로 안 삐져나오고 제대로 채웠다
        return 1;
 
    if (N >= n * m)
        return 0;
 
    int &fin = dp[N][S];//참조형 변수->시간 단축용!
    if (fin != -1)
        return fin;
 
    fin = 0;
 
    if ((S & 1) == 1)//이미 N번째 칸이 채워져 있는 경우
    {
        fin = go(N + 1, S >> 1);
        fin %= 9901;
    }
    else
    {
        //2*1(세로)을 넣는 경우
        //N+m칸 비트를 켠 뒤 N+1번째 칸으로 이동
        fin = go(N + 1, (S | (1 << m)) >> 1);
        fin %= 9901;
 
        //1*2(가로)를 넣는 경우->다음 칸이 비어 있는지 검사
        //N번째 칸이 가장 오른쪽 칸이면 1*2를 넣을 수 없으므로 그것도 검사
        if ((S & 2) == 0 && (N%m) + 1 != m)
        {
            fin += go(N + 2, S >> 2);//모든 경우의 수를 구하는 것이므로 max가 아니라 더함
            fin %= 9901;
        }
    }
 
    return fin % 9901;
 
 
 
}
 
int main()
{
    memset(dp, -1, sizeof(dp));
    scanf("%d %d", &n, &m);
    printf("%d", go(0, 0));
}
